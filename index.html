03 / Projects

Things I've Built

Project 01
SceneSolver — AI Crime Detection System
Live ↗ GitHub ↗

Manual video surveillance review is slow and misses incidents. SceneSolver is an end-to-end system that analyzes video in real-time — integrating TimeSformer for activity recognition, CLIP for context understanding, and YOLOv8 for object detection, with a React dashboard for upload and flagged event display.

  • Built the backend API, model integration pipeline, and video processing architecture using Python & Flask
  • Integrated TimeSformer, CLIP, and YOLOv8 for multi-model real-time video inference
  • Developed the React frontend dashboard for video upload and flagged event visualization
  • Solved real-time inference speed via frame sampling and background task queuing
  • Processes 10-second clips in under 4 seconds — detects fighting, vandalism, and suspicious activity
  • Deployed full-stack application on Vercel and Render
